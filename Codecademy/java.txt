- files end in .java 

Print statements:
    - System - built in Java class
    - out - short for output
    - println() - print on new line
    - print() - prints on the same line
    * System.out.println("Place words or variables to print here");

Commenting in Java:
    - // for single line comments
    - /* */ for multi line comments
    - /** */ for Java documentation comments

**All Java statements end in a semicolon ;**

Whitespace in Java doe not affect the program, so use Whitespace to enhance readability

runing "javac filename.java" runs the compiler and shows errors in compiling

once compiled, run java filename to run the complied class
